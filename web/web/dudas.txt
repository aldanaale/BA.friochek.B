Comandos para encender la aplicación
1. Asegúrate que LocalDB esté corriendo:
# Verificar que LocalDB existe y está corriendo
sqllocaldb info MSSQLLocalDB

# Si no existe, créalo:
sqllocaldb create MSSQLLocalDB

# Si está parado, arráncalo:
sqllocaldb start MSSQLLocalDB
2. Ejecutar la aplicación:
# Desde la raíz del proyecto
cd "C:\Users\ygnac\OneDrive\Escritorio\BA.friochek.B-main"

# Configurar variable JWT
$env:JWT_SECRET_KEY="TuClaveSecreta12345678901234567890"

# Modo simulación (sin BD real)
dotnet run --project src/BA.Backend.WebAPI

# O con BD real (primero configura UseRealDatabase: true en appsettings.json)
dotnet ef database update --project src/BA.Backend.Infrastructure --startup-project src/BA.Backend.WebAPI
dotnet run --project src/BA.Backend.WebAPI
3. Verificar que funciona:
# En otra terminal, prueba el endpoint
Invoke-WebRequest -Uri "http://localhost:5003/swagger" -UseBasicParsing
Resumen rápido:
sqllocaldb start MSSQLLocalDB
